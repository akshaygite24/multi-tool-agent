# Multi-Tool Agent

## Overview
This is a simple agent built using Langchain and Groq.</br>
It can use different tools to answer user queries.</br>
This is a learning Project.</br>
Focused on understanding how agents and tools work.

## Features
- Calculator (Math Operation)
- String lenght (Character Count)
- Word Count

## Example Queries
- What is 10+5?
- Length of 'Hello'
- How many words in "I am learning AI"
